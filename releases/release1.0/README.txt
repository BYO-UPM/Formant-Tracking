Formant Tracking Toolbox
________________________

Authors: Daniel Rudoy, Daniel Spendley, Patrick J. Wolfe


Formant Tracking software following the article: "Conditionally linear Gaussian models for tracking of vocal tract resonances" by 
Daniel Rudoy, Daniel Spendley and Patrick J. Wolfe appearing in INTERSPEECH 2007. The main routine is located in the run folder: runWave.m, detailed instructions and documentation may be found
inside that file. Example data sets are located in the data directory.


Direct Questions/Comments to Daniel Rudoy, rudoy@seas.harvard.edu
